## Como os clientes fazem uma encomenda

1. O cliente entra no site e navega pelo catálogo
2. Clica **"+ Adicionar"** nos produtos que quer
3. Clica no ícone do carrinho (canto superior direito)
4. Clica **"Enviar Pedido via WhatsApp"**
5. O WhatsApp abre com a mensagem já preenchida — o cliente só envia
6. Recebe o pedido no seu WhatsApp e combina a entrega

## Como usar o Painel de Gestão

Acede em: `https://so-vendas.netlify.app/admin.html`  
Senha: `hamilton1999`

### Adicionar um produto novo
1. Clica no menu **Produtos** (barra lateral esquerda)
2. Clica **"+ Novo Produto"**
3. Preenche: Nome, Categoria, Preço, Descrição
4. Clica na área de imagem para carregar a foto do produto
5. Clica **"Guardar Produto"**
6. O produto aparece no site imediatamente

### Editar ou apagar um produto
1. Menu **Produtos** → clica **Editar** no produto pretendido
2. Altera o que precisar → **Guardar Produto**
3. Para apagar → clica **Apagar** (pede confirmação)

### Marcar produto como indisponível
1. Menu **Produtos** → **Editar** → campo **Disponível** → selecciona **"Não — Indisponível"**
2. O produto aparece no site com a etiqueta "Indisponível" (não pode ser adicionado ao carrinho)

### Alterar o número WhatsApp
1. Menu **Loja** (barra lateral)
2. Campo **Número WhatsApp para Pedidos**
3. Escreve o número sem espaços nem sinais (ex: `351924148088`)
4. Clica **Guardar Configurações**

### Alterar a senha do painel
1. Menu **Loja** → secção **Segurança**
2. Escreve a nova senha (mínimo 6 caracteres)
3. Confirma a senha → **Alterar Senha**

### Melhorar o SEO (aparecer no Google)
1. Menu **SEO** (barra lateral)
2. Preenche o **Título da Página** (50–60 caracteres)
3. Preenche a **Meta Descrição** (150–160 caracteres)
4. Adiciona uma **Imagem de Partilha** (aparece quando partilhas o link no WhatsApp/Facebook)
5. Clica **Guardar SEO**
6. A pontuação SEO actualiza automaticamente (0–100)

## Perguntas frequentes

**Os dados dos produtos são guardados onde?**  
No próprio browser (localStorage). Ficam guardados mesmo que feches o browser. Se mudares de computador, precisas de re-adicionar os produtos no novo computador — ou pede ao Wesley para configurar uma base de dados.

**O site actualiza em tempo real?**  
Sim. Qualquer alteração feita no painel aparece no site imediatamente após guardar.

**Como partilhar o site?**  
Envia o link `https://so-vendas.netlify.app` por WhatsApp, Instagram, Facebook ou email.

**Posso ter um domínio próprio (ex: sovendas.pt)?**  
Sim. O registo de um domínio custa ~€10–15/ano. Contacta o Wesley para configurar.
