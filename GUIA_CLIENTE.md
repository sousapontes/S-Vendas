## Como os clientes fazem uma encomenda

1. O cliente entra no site pelo telemóvel ou computador
2. Usa a **barra de pesquisa** para procurar um produto pelo nome, ou usa os **filtros** para navegar:
   - Clica em **Roupas**, **Sapatos** ou **Pulseiras**
   - Depois pode filtrar por **Masculino**, **Feminino** ou **Criança**
3. Clica **"+ Adicionar"** nos produtos que quer (o botão muda para "✓ Adicionado")
4. Clica no **ícone do carrinho** (canto superior direito) para ver o resumo
5. Ajusta quantidades se necessário e vê o total
6. Clica **"Enviar Pedido via WhatsApp"** — a mensagem é gerada automaticamente
7. O WhatsApp abre com tudo preenchido — o cliente só carrega Enviar
8. Recebe o pedido e combina entrega e pagamento

**No telemóvel:** o menu de navegação abre com o botão ☰ no canto superior esquerdo. Os filtros deslizam horizontalmente com o dedo.

## Como usar o Painel de Gestão

Acede em: `https://so-vendas.netlify.app/admin.html`  
Senha: `hamilton1999`

### Adicionar um produto novo
1. Menu **Produtos** → **"+ Novo Produto"**
2. Preenche:
   - **Nome** do produto
   - **Categoria**: Roupa, Sapato ou Pulseira
   - **Subcategoria** (para Roupas e Sapatos): Masculino, Feminino ou Criança
   - **Preço** em euros (€)
   - **Descrição** curta do produto
3. Clica na área de imagem para carregar a foto (JPG/PNG, máx. 2MB)
4. Clica **"Guardar Produto"** — aparece no site imediatamente

> A subcategoria é importante: permite que os clientes filtrem por "Roupas Masculinas", "Sapatos Femininos", etc. As Pulseiras não precisam de subcategoria.

### Editar um produto
Painel → Produtos → **Editar** → altera o necessário → **Guardar Produto**

### Apagar um produto
Painel → Produtos → **Apagar** → confirmar (ação irreversível)

### Marcar produto como indisponível (sem apagar)
Painel → Produtos → Editar → campo **"Disponível"** → seleccionar **"Não — Indisponível"** → Guardar  
O produto aparece no catálogo com a etiqueta "Indisponível" mas não pode ser adicionado ao carrinho.

### Alterar o número WhatsApp
1. Menu **Loja** → campo **"Número WhatsApp para Pedidos"**
2. Escreve o número sem espaços nem sinais (ex: `351924148088`)
3. Clica **Guardar Configurações**

### Alterar a senha do painel
1. Menu **Loja** → secção **Segurança**
2. Escreve a nova senha (mínimo 6 caracteres) e confirma
3. Clica **Alterar Senha** — guarda a nova senha num lugar seguro

### Melhorar o SEO (aparecer no Google)
1. Menu **SEO**
2. Preenche o **Título da Página** (50–60 caracteres)
3. Preenche a **Meta Descrição** (150–160 caracteres)
4. Adiciona uma **Imagem de Partilha** (aparece quando o link é partilhado no WhatsApp/Facebook)
5. Clica **Guardar SEO**
6. A pontuação SEO actualiza automaticamente (objectivo: 80+)

> Um site novo demora 2 a 4 semanas a aparecer no Google. É normal.

## Perguntas frequentes

**Os dados dos produtos são guardados onde?**  
No browser do computador que usa para gerir o site. Se mudar de computador, precisa de re-adicionar os produtos. Para ter os dados na nuvem (acessíveis de qualquer dispositivo), contacte o Wesley.

**Como partilhar o site com os clientes?**  
Envie o link `https://so-vendas.netlify.app` por WhatsApp, Instagram, Facebook ou coloque-o na bio do Instagram.

**Posso ter um domínio próprio (ex: sovendas.st)?**  
Sim. O registo custa ~€10–15/ano. Contacte o Wesley para configurar.

**Como o cliente filtra por "Roupas Masculinas"?**  
Clica em "Roupas" nos filtros → aparece uma segunda barra com Masculino, Feminino, Criança → clica em "Masculino". Os produtos filtram automaticamente.
